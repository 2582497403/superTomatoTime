# superTomatoTime

一个强大的桌面番茄工作法应用，结合待办事项管理和智能提醒功能，帮助您提高工作效率和时间管理能力。

## 功能特性

### 🍅 番茄计时器
- **标准番茄工作法**：支持25分钟工作 + 5分钟短休息 + 15分钟长休息的经典模式
- **自定义计时**：可根据个人习惯调整工作和休息时长
- **智能提醒**：计时结束时自动播放铃声并显示通知
- **一键操作**：开始、暂停、重置计时器的便捷控制

### 📝 待办事项管理
- **任务分类**：支持不同类型的任务（工作、学习、生活等）
- **任务状态**：标记任务为进行中、已完成或待办
- **任务编辑**：轻松添加、修改、删除任务
- **完成统计**：查看每日/每周任务完成情况

### 🔔 智能提醒
- **定时提醒**：设置固定时间提醒事项
- **自定义提醒**：支持文本提醒和音频提醒
- **系统通知**：集成操作系统原生通知系统
- **提醒历史**：查看所有提醒记录

### 📊 数据统计与历史
- **计时历史**：记录每次番茄计时的详细数据
- **任务历史**：保存已完成任务的历史记录
- **数据可视化**：直观的图表展示工作效率
- **导出功能**：支持数据导出和备份

### ⚙️ 个性化设置
- **主题切换**：支持明暗主题切换
- **铃声选择**：多种铃声选项
- **通知设置**：自定义通知行为
- **数据管理**：导入/导出个人数据

## 技术栈

- **前端**：Vue 3 + Composition API
- **桌面应用**：Tauri (Rust)
- **数据库**：SQLite
- **音频处理**：Web Audio API
- **通知系统**：系统原生通知 + 自定义通知

## 安装与运行

### 环境要求
- Node.js 16+
- Rust 1.60+
- Windows/macOS/Linux

### 安装步骤
1. 克隆项目
   ```bash
   git clone <repository-url>
   cd reminder
   ```

2. 安装前端依赖
   ```bash
   npm install
   ```

3. 运行开发版本
   ```bash
   npm run dev
   ```

4. 构建桌面应用
   ```bash
   npm run build
   npm run tauri build
   ```

## 使用指南

1. **开始使用**：首次运行应用，设置您的偏好设置
2. **创建任务**：在待办事项面板中添加您的任务
3. **开始计时**：选择任务并开始番茄计时
4. **专注工作**：跟随计时器完成工作周期
5. **查看统计**：在历史面板中查看您的进度
